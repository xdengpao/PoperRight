# 右侧趋势突破选股策略操作手册

> 策略类型：右侧趋势跟踪 | 交易周期：中短线（3-15 个交易日） | 适用市场：A 股主板/创业板

---

## 1. 策略概述

本策略面向右侧交易者，核心逻辑为 **趋势确认 + 形态突破 + 板块共振 + 量价验证 + 资金共振**，采用八因子 AND 逻辑严格过滤，只在个股趋势与板块趋势双重确立后介入，避免左侧抄底和孤立突破风险。

### 1.1 策略特征

- 八因子全部满足才出信号，信号频率较低但胜率高
- 个股突破 + 板块趋势双重确认，过滤板块不配合的孤立突破
- 板块涨幅排名过滤，确保个股处于强势板块有资金接力
- 主力资金流入验证，确保有资金推动
- RSI 强势区间过滤，排除超买追高

### 1.2 适用场景

| 场景 | 适用性 |
|------|--------|
| 牛市/震荡偏强 | 最佳，信号充足 |
| 震荡市 | 适用，可适当收紧参数 |
| 熊市/单边下跌 | 不适用，应暂停策略 |

## 2. 因子配置详解

### 2.1 八因子一览

| 序号 | 因子 | 条件 | 权重 | 右侧交易含义 |
|------|------|------|------|-------------|
| 1 | ma_trend | ≥ 75 | 20% | 均线多头排列，确认中期趋势向上 |
| 2 | breakout | = True | 20% | 箱体/前高/趋势线突破，量比 ≥ 1.5 倍确认 |
| 3 | sector_rank | ≤ 25 | 12% | 所属板块涨幅排名前 25，确认处于强势板块 |
| 4 | sector_trend | = True | 8% | 板块均线多头排列，提供趋势级别保护 |
| 5 | macd | = True | 12% | MACD 零轴上方金叉，红柱放大确认动能 |
| 6 | turnover | 3%-15% | 8% | 换手率适中，排除冷门股和过度投机股 |
| 7 | money_flow | ≥ 75 | 12% | 主力资金净流入排名前 25%，确认资金参与 |
| 8 | rsi | 55-80 | 8% | RSI 强势但未超买，仍有上行空间 |

因子分为三层验证体系：
- **趋势层**（ma_trend + breakout）：个股自身趋势与突破确认，权重合计 40%
- **板块层**（sector_rank + sector_trend）：板块环境验证，权重合计 20%
- **量价资金层**（macd + turnover + money_flow + rsi）：动能与资金确认，权重合计 40%

### 2.2 因子逻辑

- 逻辑运算：**AND**（全部满足才入选）
- 评分公式：`综合得分 = Σ(因子得分 × 因子权重)`
- 结果排序：按 `trend_score` 降序排列

## 3. 平台配置步骤

### 3.1 创建策略模板

1. 进入 **智能选股** 页面（`/screener`）
2. 点击「新建策略」或从示例库加载后修改
3. 按以下 JSON 配置因子条件：

```json
{
  "factors": [
    {
      "factor_name": "ma_trend",
      "operator": ">=",
      "threshold": 75,
      "params": {}
    },
    {
      "factor_name": "breakout",
      "operator": "==",
      "threshold": null,
      "params": {}
    },
    {
      "factor_name": "sector_rank",
      "operator": "<=",
      "threshold": 25,
      "params": {}
    },
    {
      "factor_name": "sector_trend",
      "operator": "==",
      "threshold": null,
      "params": {}
    },
    {
      "factor_name": "macd",
      "operator": "==",
      "threshold": null,
      "params": {}
    },
    {
      "factor_name": "turnover",
      "operator": "BETWEEN",
      "threshold": null,
      "params": { "threshold_low": 3.0, "threshold_high": 15.0 }
    },
    {
      "factor_name": "money_flow",
      "operator": ">=",
      "threshold": 75,
      "params": {}
    },
    {
      "factor_name": "rsi",
      "operator": "BETWEEN",
      "threshold": null,
      "params": { "threshold_low": 55, "threshold_high": 80 }
    }
  ],
  "logic": "AND",
  "weights": {
    "ma_trend": 0.20,
    "breakout": 0.20,
    "sector_rank": 0.12,
    "sector_trend": 0.08,
    "macd": 0.12,
    "turnover": 0.08,
    "money_flow": 0.12,
    "rsi": 0.08
  },
  "enabled_modules": ["ma_trend", "breakout", "indicator_params", "volume_price"]
}
```

4. 启用模块勾选：`ma_trend`、`breakout`、`indicator_params`、`volume_price`

### 3.2 模块参数配置

#### 均线趋势模块（ma_trend）

| 参数 | 值 | 说明 |
|------|-----|------|
| ma_periods | [5, 10, 20, 60, 120] | 参与计算的均线周期 |
| trend_score_threshold | 68 | 初选池纳入阈值 |
| support_ma_lines | [20, 60] | 均线支撑检测线 |

#### 形态突破模块（breakout）

| 参数 | 值 | 说明 |
|------|-----|------|
| box_breakout | true | 启用箱体突破检测 |
| high_breakout | true | 启用前期高点突破检测 |
| trendline_breakout | true | 启用下降趋势线突破检测 |
| volume_ratio_threshold | 1.5 | 突破量比倍数阈值 |
| confirm_days | 1 | 站稳确认天数 |

#### 技术指标模块（indicator_params）

| 参数 | 值 | 说明 |
|------|-----|------|
| macd_fast / slow / signal | 12 / 26 / 9 | MACD 标准参数 |
| rsi_period | 14 | RSI 计算周期 |
| rsi_lower / rsi_upper | 55 / 80 | RSI 强势区间 |

#### 量价资金模块（volume_price）

| 参数 | 值 | 说明 |
|------|-----|------|
| turnover_rate_min / max | 3.0 / 15.0 | 换手率区间 (%) |
| main_flow_threshold | 1000.0 | 主力净流入阈值（万元） |
| main_flow_days | 2 | 连续净流入天数 |
| large_order_ratio | 30.0 | 大单占比阈值 (%) |
| min_daily_amount | 5000.0 | 日均成交额下限（万元） |

#### 板块筛选配置（sector_config）

| 参数 | 值 | 说明 |
|------|-----|------|
| sector_data_source | DC | 东方财富概念板块（覆盖面广、更新及时） |
| sector_period | 3 | 板块涨幅计算周期（天），3 天更贴合右侧短期动量 |
| sector_top_n | 25 | 板块排名阈值，前 25 名视为强势板块 |

> 板块数据源选择说明：DC（东方财富）概念板块覆盖面最广，适合捕捉热点轮动；如需行业维度验证，可切换为 TI（申万行业）。

## 4. 选股执行流程

### 4.1 盘后选股（每日 15:30 后）

```
步骤 1：进入 /screener 页面，选择已保存的「右侧趋势突破综合策略」模板
步骤 2：执行 EOD（盘后）选股
步骤 3：进入 /screener/results 查看结果
步骤 4：按 trend_score 降序排列，筛选候选股
步骤 5：将候选股加入 /stock-pool 选股池查看详情
```

### 4.2 候选股筛选标准

优先选择：
- `trend_score` ≥ 80（趋势强度高）
- `has_fake_breakout` = false（非假突破）
- 信号新鲜度为 `NEW`（当日新出现的突破信号）
- 信号强度为 `STRONG`
- 所属板块近 3 日涨幅排名持续上升（板块动量加速）

排除条件：
- 大盘风险等级为 `DANGER` 时不开新仓
- 风控标记为 `HIGH` 风险的个股
- 近期有 ST 或退市风险的个股
- 所属板块已连续上涨超过 5 日且涨幅超过 15%（板块过热风险）

### 4.3 盘前确认（次日 9:00 前）

1. 检查隔夜是否有重大利空消息
2. 确认大盘风险等级（`/dashboard` 页面）
3. 复核候选股是否仍满足策略条件
4. 确定买入数量和仓位分配

## 5. 买入操作

### 5.1 买入时机

| 时段 | 操作 |
|------|------|
| 9:15-9:25 集合竞价 | 观察竞价量价，高开 1%-3% 且量能放大为佳 |
| 9:30-10:00 开盘半小时 | 主要买入窗口，确认突破位站稳后挂单 |
| 10:00 后 | 若错过早盘窗口，观察回踩突破位是否获支撑再介入 |

### 5.2 买入价格

- 挂单价格：突破位上方 1%-2% 的限价单
- 买入参考价：以选股结果中的 `ref_buy_price` 为基准
- 避免追涨：若开盘涨幅已超过 5%，放弃本次买入

### 5.3 仓位控制

| 规则 | 限制 |
|------|------|
| 单票仓位 | 不超过总资金的 15% |
| 同板块仓位 | 不超过总资金的 30% |
| 最大持仓数 | 不超过 10 只 |
| 总仓位上限 | NORMAL 状态下不超过 80% |

## 6. 持仓管理与止损

### 6.1 止损策略（在 /risk 页面配置）

推荐使用 ATR 自适应止损模式：

```json
{
  "mode": "atr_adaptive",
  "fixed_stop_loss": 8.0,
  "trailing_stop": 5.0,
  "trend_stop_ma": 20,
  "atr_fixed_multiplier": 2.0,
  "atr_trailing_multiplier": 1.5
}
```

### 6.2 分阶段持仓管理

| 阶段 | 触发条件 | 操作 |
|------|---------|------|
| 买入当日 | 建仓完成 | 设置固定止损 = 买入价 × 92%（-8%） |
| 盈利 5%+ | 浮盈达到 5% | 切换移动止盈，回撤 5% 触发卖出 |
| 盈利 10%+ | 浮盈达到 10% | 收紧移动止盈，回撤 3% 触发卖出 |
| 持仓 15 日+ | 超过 15 个交易日 | 评估 ma_trend，跌破 60 则减仓 |
| 趋势破位 | 收盘价跌破 20 日均线 | 无条件止损，次日开盘卖出 |

### 6.3 卖出信号

以下任一条件触发时执行卖出：
- 固定止损触发（亏损达 8%）
- 移动止盈触发（从最高点回撤 5%/3%）
- 趋势破位（收盘跌破 20 日均线）
- ma_trend 连续 2 日下降超过 10 分
- 主力资金连续 3 日净流出
- 所属板块排名跌出前 50 且板块趋势转空（板块退潮信号）

## 7. 每日复盘

### 7.1 复盘检查清单（/review 页面）

- [ ] 持仓股 ma_trend 是否仍 ≥ 65（趋势延续）
- [ ] 持仓股 money_flow 是否仍为正（资金未撤退）
- [ ] 持仓股所属板块排名是否仍在前 30（板块未退潮）
- [ ] 持仓股所属板块趋势是否仍为多头（板块趋势未反转）
- [ ] 大盘风险等级是否变化
- [ ] 是否有新的选股信号出现
- [ ] 止损/止盈条件是否需要调整

### 7.2 策略健康度评估

每周末对策略进行回顾：

| 指标 | 健康标准 | 异常处理 |
|------|---------|---------|
| 周胜率 | ≥ 50% | 低于 40% 连续 2 周，暂停策略 |
| 盈亏比 | ≥ 1.5:1 | 低于 1:1 时检查止损是否过宽 |
| 最大回撤 | ≤ 15% | 超过 10% 时降低仓位至 50% |
| 信号频率 | 每周 3-8 个 | 过少则放宽参数，过多则收紧 |

## 8. 市场环境适配

### 8.1 大盘风控联动

| 大盘状态 | 判定条件 | 策略调整 |
|---------|---------|---------|
| NORMAL | 趋势打分 ≥ 80 | 正常执行，总仓位可用 80% |
| CAUTION | 跌破 20 日均线 | ma_trend 阈值提高至 85，仓位降至 50% |
| DANGER | 跌破 60 日均线 | 暂停新开仓，仅管理现有持仓 |

### 8.2 参数调优建议

| 市场环境 | ma_trend | money_flow | rsi 区间 | 换手率区间 | sector_rank | sector_period |
|---------|----------|------------|---------|-----------|-------------|---------------|
| 强势牛市 | ≥ 70 | ≥ 70 | [50, 85] | [3%, 20%] | ≤ 30 | 5 天 |
| 震荡市（默认） | ≥ 75 | ≥ 75 | [55, 80] | [3%, 15%] | ≤ 25 | 3 天 |
| 弱势震荡 | ≥ 85 | ≥ 80 | [60, 75] | [3%, 12%] | ≤ 15 | 3 天 |

## 9. 回测验证

### 9.1 回测建议

在 `/backtest` 页面执行回测验证：

| 参数 | 建议值 |
|------|--------|
| 回测周期 | 近 6 个月至 1 年 |
| 初始资金 | 100 万 |
| 单股最大仓位 | 15% |
| 最大持仓数 | 10 |
| 止损比例 | 8% |
| 移动止盈回撤 | 5% |

### 9.2 关注指标

- 胜率（目标 ≥ 50%）
- 盈亏比（目标 ≥ 1.5:1）
- 夏普比率（目标 ≥ 1.0）
- 最大回撤（目标 ≤ 15%）
- 卡玛比率（目标 ≥ 1.0）
- 平均持仓天数（预期 5-12 天）

## 10. 风险提示

1. 本策略为右侧趋势跟踪策略，在趋势反转初期可能产生亏损
2. 八因子 AND 逻辑较为严格，信号频率较低，需耐心等待
3. 形态突破存在假突破风险，系统已内置假突破检测但无法完全避免
4. 板块热度消退后个股可能快速回落，需关注板块排名变化
5. 板块数据源（DC/TI/TDX）之间可能存在数据延迟差异
6. 主力资金流数据存在一定滞后性，盘后数据更为可靠
7. 任何策略都无法保证盈利，请严格执行止损纪律
8. 建议先用模拟交易（PAPER 模式）验证 1-2 周后再实盘操作
