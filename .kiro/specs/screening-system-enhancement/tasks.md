# Implementation Plan: 智能选股系统增强

## Overview

本实现计划将 22 项需求分解为可增量执行的编码任务，涵盖：
1. 因子注册表扩展（19→52 个因子，新增 3 个 FactorCategory）
2. 板块面因子分类重构（SectorScreenConfig 移除 sector_type 必选）
3. 数据提供服务扩展（6 个新 `_enrich_*_factors()` 批量方法）
4. 现有系统修复（资金流接入、Celery 任务、板块因子、风控集成、评分重构等）
5. 策略示例库扩展（12→22 个，含配置说明书）
6. 前端因子编辑器增强（新类别、数据来源下拉）
7. API 层适配

所有新增代码注释和文档字符串使用中文，标注对应需求编号。
